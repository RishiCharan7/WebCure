# WebCure
WebCure was created to bridge this gap. It is a zero-installation, zero-cost, fully browser-accessible vulnerability scanner that performs automated checks across the OWASP Top 10 (2021) — the globally recognized standard for web application security risks — and communicates findings in plain English with specific, beginner-friendly fix instructions. Whether the user is a professional developer running a quick audit or a non-technical business owner wondering if their website is safe, WebCure delivers a complete, understandable security assessment.
